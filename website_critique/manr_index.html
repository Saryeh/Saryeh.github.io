Man Repeller
Man Repeller is a website dedicated to talking about fashion and subjects that havent been delved into before. The website mimics the founder, Leandra Cohen's style and aesthetic however the ads get in the way of appreciating the layout and images. The website is dense with so much content and is so frequenly interuppted by ads that I can never get a chance to view the webpage as a whole. If Man Repeller could improve the layout so the ads dont get in the way of viewing the content that would make the browsing process more pleasant.
